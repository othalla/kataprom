# kataprom


## Prerequisite

Python 3.6 + installed

## Install & Usage


```shell
git clone https://github.com/othalla/kataprom.git && cd kataprom
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python kataprom/app.py
```

## Use IT

Access it through http://localhost:8080/v0/ui
